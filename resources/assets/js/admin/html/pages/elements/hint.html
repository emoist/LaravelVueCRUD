{% markdown %} Hint is built with pure css code. It can be apply on any elements who's `overflow` is not hidden and `position` is not undefined. ```html ``` {% endmarkdown %}

Sizes

{% markdown %} By default, Hint will always remain single-line. you can change this behavior by adding `.hint--sm`, `.hint--md` or `.hint--lg` modifier class. ```html ``` {% endmarkdown %}

Position

{% markdown %} Hint use top as default position. By adding `.hint--right`, `.hint--bottom` or `.hint--left` modifier class to change hint position. ```html ``` {% endmarkdown %}
