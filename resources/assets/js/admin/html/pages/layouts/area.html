Area

{% markdown %} Since [Grid Layout](https://css-tricks.com/snippets/css/complete-guide-grid/) is still a experimental feature, so we create **Area** layout component, which is a simple simulation of Grid. Combinng 12-grid system and **Area**, you can create a powerful responsive layout. The **Area** container element will set it's child position to absolute and fill full of the container. Use `.l-area@` to create **Area** only at certain breakpoint. `.l-area--MxN`, `M` and `N` are both range 1 through 12, and `M` shound has the same value of it parent column. {% endmarkdown %}
